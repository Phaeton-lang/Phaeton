◆ ph_unreachable

#define ph_unreachable (   msg)    ::phaeton::ph_unreachable_internal(msg, __FILE__, __LINE__)

Prints the message and location info to stderr in !NDEBUG builds. This is intended to be used for "impossible" situations that imply a bug in the compiler.

In NDEBUG mode it only prints "UNREACHABLE executed". Use this instead of assert(0), so that the compiler knows this path is not reachable even for NDEBUG builds.
